# QGIS Entity Attributes

## Room
### At least one of the following: 
- number
- name

## Room entrance
### Required:
- room (corresponds to either the room number or name - has to be exact)
### Optional: 
- name

## Building
### Required: 
- name 

## Path
### Optional:
- name

## Landing
### Required: 
- staircase (index in the list of staircases)
